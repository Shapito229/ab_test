# ab_test
Необходимо провести оценку результатов A/B-теста. В вашем распоряжении есть датасет с действиями пользователей, техническое задание и несколько вспомогательных датасетов.

Оценить корректность проведения теста и проанализируйте его результаты.

Чтобы оценить корректность проведения теста:

удостоверьтесь, что нет пересечений с конкурирующим тестом и нет пользователей, участвующих в двух группах теста одновременно;
проверьте равномерность распределения пользователей по тестовым группам и правильность их формирования.
